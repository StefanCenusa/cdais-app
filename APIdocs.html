Groups 

Groups CRUD 

Groups are groups of users which are usually created with association to debate events - people that attend an event are members of a group. Therefore, they can receive notifications and other materials releated to that group. However, we do not enforce the rule that groups should be associated with events. If you need to, you can use the differently.
